Servicios en Angular

Los **Servicios** son clases que encapsulan lógica de negocio reutilizable y proporcionan funcionalidad a los componentes. Son ideales para tareas como la comunicación con APIs externas, la gestión de datos compartidos o cualquier lógica que no esté directamente ligada a la interfaz de usuario. Gracias a la **Inyección de Dependencias**, los servicios pueden ser fácilmente proporcionados y consumidos por los componentes.

Un servicio generalmente se marca con el **Decorador `@Injectable()`**, lo que indica que puede ser inyectado en otras clases y que el inyector de dependencias de Angular puede crear una instancia de él.

Ejemplo de un Servicio Básico y su Consumo:

<!-- data.service.ts --> import { Injectable } from '@angular/core'; <!-- El decorador @Injectable() marca una clase como un servicio que puede ser inyectado. --> <!-- El metadato 'providedIn: 'root'' significa que el servicio es un singleton y se --> <!-- proporciona en el inyector raíz, haciéndolo disponible en toda la aplicación. --> @Injectable({ providedIn: 'root' }) export class DataService { private _data: string[] = ['Item 1', 'Item 2', 'Item 3']; constructor() { } getData(): string[] { return this._data; } addData(item: string): void { this._data.push(item); } }

Para consumir este servicio en un componente, se inyecta en el constructor del componente:

<!-- service-explanation.component.ts (parte relevante) --> import { Component, OnInit } from '@angular/core'; import { DataService } from '../../services/data.service'; <!-- Asegúrate de la ruta correcta --> @Component({ selector: 'app-service-explanation', templateUrl: './service-explanation.component.html', styleUrls: ['./service-explanation.component.css'] }) export class ServiceExplanationComponent implements OnInit { dataItems: string[] = []; newItem: string = ''; <!-- Inyección de dependencias: Angular inyecta una instancia de DataService --> <!-- cuando crea una instancia de ServiceExplanationComponent. --> constructor(private dataService: DataService) { } <!-- ngOnInit es un Hook del ciclo de vida de Angular. --> <!-- Se ejecuta una vez que Angular ha inicializado todas las propiedades en el componente. --> ngOnInit(): void { this.dataItems = this.dataService.getData(); } addItem(): void { if (this.newItem.trim()) { this.dataService.addData(this.newItem); this.dataItems = this.dataService.getData(); <!-- Actualiza la lista --> this.newItem = ''; <!-- Limpia el input --> } } }
